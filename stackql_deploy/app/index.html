§StackQL Deploy Application Constants

This module defines various constants and configuration values for the StackQL Deploy application. It includes general application metadata, default settings, supported providers, and paths to templates.

§Usage Example

use crate::app::{APP_NAME, APP_VERSION, DEFAULT_SERVER_HOST, DEFAULT_SERVER_PORT};

println!("{} v{} running on {}:{}",
    APP_NAME, APP_VERSION, DEFAULT_SERVER_HOST, DEFAULT_SERVER_PORT
);

This module also contains sub-modules for template-related constants specific to AWS, Azure, and Google platforms.
