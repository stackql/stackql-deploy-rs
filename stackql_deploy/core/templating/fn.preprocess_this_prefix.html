Expand description

Pre-process this. prefix inside Tera template blocks.

Within every {{ ... }} and {% ... %} block, replaces this. with {resource_name}., allowing resource-scoped variables to be referenced unambiguously inside a resource’s own .iql file.

Returns Err with a diagnostic if this. appears but resource_name is empty (i.e. no active resource context, such as a global template).
