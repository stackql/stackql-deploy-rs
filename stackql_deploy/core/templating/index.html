§Templating Module

Handles loading, parsing, and rendering SQL query templates from .iql files. Matches the Python lib/templating.py implementation.

Queries are loaded and parsed eagerly, but rendered lazily (JIT) when actually needed. This avoids errors from templates that reference variables not yet available in the context (e.g., delete queries referencing exports that haven’t been computed yet during a build operation).
