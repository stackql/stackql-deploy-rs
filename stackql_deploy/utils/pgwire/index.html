Pure-Rust PostgreSQL simple-query wire protocol client.

Implements only what stackql-deploy needs: unencrypted TCP connections to a local StackQL server using the PostgreSQL simple query protocol (v3). No native dependencies (replaces pgwire-lite → libpq-sys).
