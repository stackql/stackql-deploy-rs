3//! # StackQL Deploy Application Constants
4//!
5//! This module defines various constants and configuration values for the StackQL Deploy application.
6//! It includes general application metadata, default settings, supported providers, and paths to templates.
7//!
8//! ## Usage Example
9//! ```rust
10//! use crate::app::{APP_NAME, APP_VERSION, DEFAULT_SERVER_HOST, DEFAULT_SERVER_PORT};
11//!
12//! println!("{} v{} running on {}:{}",
13//!     APP_NAME, APP_VERSION, DEFAULT_SERVER_HOST, DEFAULT_SERVER_PORT
14//! );
15//! ```
16//!
17//! This module also contains sub-modules for template-related constants specific to
18//! AWS, Azure, and Google platforms.
19
